---
name: copyedit
description: Copy-edit a post -- reviewing for typographical or other local errors.
---

When copy-editing a post, your job is to identify (and fix, where possible) small, local errors. You're looking for errors such as:

- Clear typos -- misspellings, accidentally duplicated words, and so on. When it's clear what the intention was, fix these directly in the document. If it's unclear what was meant, flag it for the user instead.
- Commentary or TODOs that were left in unintentionally. I often label these "TK" or "TKTK" but flag anything suspicious. Don't automatically delete these -- flag them.
- Markdown or other syntax errors. Markdown link errors -- a `[bracketed][link]` where `[link]` is not defined in the document, and so on.

Fix errors in-place where possible. When you're unsure what was intended or something is ambiguous, flag to the user instead. Flag errors by reporting a bulleted list to the user. Each error report should contain the line number on which the error occurs.
